1. tf 卡系统烧录
1.1 准备 tf 卡
1.2 准备好 Ubuntu 24 的 io 板镜像
1.3 安装 win32diskimager
1.4 tf 卡接入到windows，格式化成 exfat 格式
1.5 打开 win32diskimager， 选择 1.2 的系统镜像，写入 tf 卡
1.6 写好系统镜像的 tf 卡插入到猫5 的插槽上，断电，再插电，即可进入 tf  卡的系统

2. emmc  系统备份
2.1 使用 1. 的 tf 卡启动系统
2.2 运行 sudo fire-config，选择从 sd 卡备份 emmc 系统
2.3 运行完成后， /home/cat/backup.img 即为系统镜像

3. 烧录 emmc 系统备份
3.1 启动 RKDevTool v2.86
3.2 选择 3688 mini loader，以及 2. 的系统镜像 backup.img
3.2 连接猫5 的 type c 口到电脑的 usb 口，断开电源，按住 maskrom 键，插入电源
3.3 RKDevTool v2.86 发现 maskrom 设备后，松开 maskrom 按键，选中程序下部的“强制按地址写入”
3.4 写入完成会自动重启， 烧录完毕
3.5 烧录重启后，必须登录一次账号，不然有线网卡分配不到 IP
